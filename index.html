Friendly Links in the Cubic Lounge

Here are some neat links and such. I'll probably keep adding to this as I go, if you have suggestions you should suggest them.

h3Dro Trilogy - A compilation of three classic 3D games. They're pretty simple, but fun nonetheless.

h3Dro Trilogy

White Dot Bullet Game - A small but addicting game made in Khan Academy's Javascript editor.

White Dot Bullet Game

Dinosaur Game - The game that appears when Chrome can't connect to the internet.

T-Rex Game

Deer Trail - A neat website with several cameras set up in the woods. Great for nature stuff, although most videos are restricted by default.

Deer Trail

Zelda 30th Anniversary Tribute - An unfinished fangame that was taken down from the internet for copyright reasons. It's a voxel-based recreation of the original game.

Zelda 30th Anniversary Tribute

Asteroids - It's Asteroids, but on this site. Pretty simple stuff.

Asteroids

Snake - A classic game of snake.

Snake

HexGL - A futuristic racing game. It's sort of like F-Zero, but without AIs. It runs surprisingly well in the browser.

HexGL

Hextris - Like tetris, but on a hexagon.

Hextris

Chat - Chat with other people! Like Google Hangouts, but... Oh, there really isn't much of an advantage.

Chat
